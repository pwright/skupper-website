skupper link create

Links this skupper site to the site that issued the token

Synopsis

Links this skupper site to the site that issued the token

skupper link create <input-token-file>  --[option]
Options
cost

Specify a cost for this link. (default 1) (int32)

name

Provide a specific name for the link (used when deleting it) (string)
